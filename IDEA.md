# ReviewBoost — AI 이커머스 리뷰 분석기

## 한 줄
리뷰 데이터를 긁어서 감정/키워드/트렌드를 대시보드로 보여주는 SaaS

## 핵심 가치
"우리 제품 리뷰에서 고객이 진짜 말하는 게 뭔지 한눈에"

## 30초 이해
쿠팡/아마존 리뷰 수백개 → "배송 불만 34%, 사이즈 문제 22%" 한방 요약

## 돈낼 의향
셀러/브랜드는 리뷰 인사이트에 목마름 → 월 3~9만 원

## 기술 스택
- Next.js + Supabase + GPT-4o-mini
- 리뷰 크롤링: 쿠팡/아마존 API or scraping

## 수익 모델
- Starter: 2.9만/월 (1브랜드, 5제품)
- Pro: 5.9만/월 (3브랜드, 20제품)
- Business: 9.9만/월 (무제한)

## MVP 평가
- 핵심가치: ⭐⭐⭐⭐ (인사이트 즉시 확인)
- 30초이해: ⭐⭐⭐⭐⭐ (대시보드 한눈에)
- 돈낼의향: ⭐⭐⭐⭐ (셀러 필수 툴)

## vs ReplyMate
- ReplyMate = 리뷰에 답글 달기 (반응형)
- ReviewBoost = 리뷰에서 인사이트 뽑기 (분석형)
- 같은 리뷰 영역, 다른 방향
